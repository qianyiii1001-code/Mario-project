# 基于 Mario BCI 运动想象实验方案框架

## 1. 实验概述

**实验名称**：基于左右手运动想象的 BCI-Mario 反馈实验

**实验目的**：采集被试在左右手运动想象（Motor Imagery, MI）过程中的 EEG 信号，通过 Mario 游戏化视觉反馈增强想象投入度，用于构建或评估 BCI 分类模型。

**实验范式**：经典 MI 范式（Cue → Prepare → MI → Feedback），融入 Mario 像素风格游戏化反馈。

---

## 2. 被试信息

| 项目 | 要求 |
|------|------|
| 人数 | 根据实验设计确定（建议 ≥ 12） |
| 年龄 | 18–65 岁 |
| 视力 | 正常或矫正至正常 |
| 运动功能 | 双手运动功能正常 |
| 排除标准 | 癫痫病史、严重认知障碍、近 24h 服用影响中枢神经药物 |

## 3. 实验设备

| 设备 | 用途 |
|------|------|
| EEG 放大器 | 采集脑电信号（如 Neuroscan, Brain Products, g.tec 等） |
| 电极帽 | 按 10-20 / 10-10 系统排布 |
| 并口 (LPT) / TTL | 发送 Event Marker，与 EEG 数据同步 |
| 刺激呈现计算机 | 运行 Mario 实验程序，分辨率自适应全屏 |
| 导电胶/电极膏 | 降低阻抗 |

---

## 4. 实验任务

### 4.1 任务描述

被试通过**运动想象**（不做实际动作）控制游戏角色吃金币：

| 想象内容 | 游戏反馈 | 视觉方向 |
|----------|----------|----------|
| 左手发力推 | 角色向左跑 → 吃左边金币 | 画面向左 |
| 右手发力推 | 角色向右跑 → 吃右边金币 | 画面向右 |

### 4.2 角色与场景

每个 Block 随机组合：

| 随机项 | 选项 |
|--------|------|
| 角色 | Mario / Peach / Luigi |
| 背景 | 经典草地 / 沙漠 / 海底 |

---

## 5. 实验流程

### 5.1 总体结构

```
开始菜单 → 指导语(2页) → Block 1 → 休息 → Block 2 → 休息 → Block 3 → 结束
```

### 5.2 单次 Trial 时间线

| 阶段 | 时长(s) | 内容 | 画面 |
|------|---------|------|------|
| **REST** | 2.0 | 静息基线 | 角色静止 + 场景 |
| **CUE** | 3.6 | 任务提示 | 文字"当前任务：左/右手想象" + 金币预览 |
| **READY** | 1.0 | 准备 | 画面显示"准备" |
| **COUNTDOWN** | 3.0 | 倒计时 | 屏幕中央 3 → 2 → 1 |
| **MI** | 4.0 | 运动想象 | 静态画面，无动画（纯想象阶段） |
| **FEEDBACK** | 4.0 | 视觉反馈 | 角色匀速跑向金币 + 吃币粒子特效 + 音效 |
| **FREEZE** | 1.2 | 结果定格 | 画面静止，金币数更新 |
| **BLINK** | 1.5 | 眨眼休息 | 提示放松 |

> **单次 Trial 总时长 ≈ 20.3 秒**

### 5.3 Block 结构

- 每 Block **20 个 trial**（10 左手 + 10 右手，伪随机打乱）
- 每 Block 时长 ≈ 20 × 20.3 ≈ **6.8 分钟**
- Block 间休息 **15 秒**
- 共 **3 个 Block**，总实验时长 ≈ **21 分钟**

### 5.4 Trigger 编码表

| Trigger | Code | 含义 |
|----------|------|------|
| `MI_START_LEFT` | 1 | 左手运动想象开始 |
| `MI_END_LEFT` | 11 | 左手运动想象结束 |
| `MI_START_RIGHT` | 2 | 右手运动想象开始 |
| `MI_END_RIGHT` | 12 | 右手运动想象结束 |
| `BLOCK_START` | 100 | Block 开始 |
| `BLOCK_END` | 101 | Block 结束 |
| `EXPERIMENT_END` | 255 | 实验结束 |

> Trigger 通过并口 LPT 发送 TTL 脉冲（15ms 脉宽），与 EEG 数据同步记录。

## 6. 实验前准备

### 6.1 环境准备
- [ ] 实验室安静、光线适宜、屏蔽电磁干扰
- [ ] 刺激呈现计算机连接并口/串口至 EEG 放大器
- [ ] 确认并口地址正确（默认 0x3FF8，可在代码中修改）
- [ ] 以管理员权限运行程序（并口驱动需要）

### 6.2 被试准备
- [ ] 签署知情同意书
- [ ] 填写被试信息表（年龄、性别、利手、健康状况等）
- [ ] 洗头、吹干（降低头皮阻抗）
- [ ] 佩戴电极帽，注入导电胶
- [ ] 确认各电极阻抗 < 5 kΩ（或根据设备要求）
- [ ] 被试坐于屏幕前约 60–80 cm，双手自然放在腿上

### 6.3 软件准备
- [ ] 确认 `inpoutx64.dll` 或 `inpout32.dll` 已放入 `C:\Windows\System32\`
- [ ] 确认 `assets/` 目录下字体、图片、音效文件完整
- [ ] 运行一次预测试，确认 Trigger 正确发送
- [ ] 检查屏幕全屏自适应是否正常

## 8. 数据采集

### 8.1 EEG 数据
- 采样率：根据放大器设置（建议 ≥ 250 Hz）
- 导联数：根据电极帽配置（建议 ≥ 32 导）
- 参考电极：根据实验设计（Cz / 耳垂 / 平均参考）
- 滤波：在线带通 0.5–40 Hz，陷波 50 Hz

### 8.2 行为数据
程序自动打印日志，包含：
- Block 编号、Trial 编号、Class ID（左=2, 右=1）
- 各阶段起止时间戳
- 角色和背景选择
- Trigger 发送记录

### 8.3 同步
- EEG 数据通过 TTL Trigger 与刺激呈现同步
- 所有 MI 起止、Block 起止均有对应 Marker

---

## 9. 数据分析管线（建议）

```
原始 EEG
  ↓ 滤波 (0.5–40 Hz BP, 50 Hz Notch)
  ↓ 伪迹去除 (ICA / 手动剔除)
  ↓ 数据分段 (以 MI_START 为 0 点, [-2s, +4s])
  ↓ 基线校正
  ↓ 特征提取 (CSP / FBCSP / 时频分析)
  ↓ 分类 (LDA / SVM / CNN / 黎曼几何)
  ↓ 交叉验证评估 (Block-wise / LOSO)
  ↓ 统计分析 (分类精度 vs chance level)
```

---

## 10. 预期结果

- 左右手 MI 在 **μ 节律 (8–13 Hz)** 和 **β 节律 (13–30 Hz)** 频段产生对侧感觉运动区事件相关去同步（ERD）
- 左手 MI → 右侧 C3 电极附近 ERD 更显著
- 右手 MI → 左侧 C4 电极附近 ERD 更显著
- 分类准确率预期 > 70%（二分类 chance level = 50%）
