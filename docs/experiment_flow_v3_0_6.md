## 时间线总览

| 阶段 | 单次时长 | ×次数 | 合计 |
|------|---------|-------|------|
| 指导语 | ~30s | 1 | ~0.5 min |
| 90s 静息 | 90s | 5 | 7.5 min |
| 单次 Trial | 13.2s | 20×5=100 | 22 min |
| Block 完成画面 | 3s | 5 | 0.25 min |
| 自评问卷 | ~20s | 5 | ~1.7 min |
| 15s 休息 | 15s+ | 5 | ~1.3 min |
| **总计（估算）** | | | **~33 min** |

## Trigger 编码

| 事件 | Code |
|------|------|
| MI_START_LEFT | 1 |
| MI_START_RIGHT | 2 |
| MI_END_LEFT | 11 |
| MI_END_RIGHT | 12 |
| BLOCK_START | 100 |
| BLOCK_END | 101 |
| EXPERIMENT_END | 255 |

## 快捷键

| 按键 | 功能 |
|------|------|
| 空格 | 确认 / 跳过等待 |
| P | 跳过当前 Block |
| R | 延长/结束延长休息（15s 休息阶段） |
| E | 终止实验 |
| ESC | 退出程序 |
