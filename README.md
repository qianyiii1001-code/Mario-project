# Super BCI Mario 🍄

基于运动想象（Motor Imagery）的脑机接口（BCI）实验平台，将经典超级马里奥游戏化元素融入神经科学实验范式。

## 项目简介

本项目通过游戏化设计，引导被试完成运动想象任务。被试通过想象左右手抓握动作来控制角色移动获取金币。实验过程中通过并口发送 TTL Trigger 信号，配合脑电（EEG）采集设备同步记录脑信号。

## 核心功能

- 🎮 **像素风马里奥游戏界面**：经典 8-bit 视觉风格，多种角色（Mario / Peach / Luigi）和场景（经典 / 沙漠 / 海底）
- 🧠 **三分类实验范式**：左手抓握 / 右手抓握 / 保持静止，含学习试次与正式实验
- ⚡ **TTL Trigger 同步**：通过并口（LPT）发送事件标记，与 EEG 采集设备精确同步
- 📊 **自评问卷**：每个 Block 结束后收集认知疲劳与身体疲劳主观评分
- 🔧 **全屏自适应**：自动适配不同分辨率屏幕，居中显示

## 实验结构（v3_0_7）

| 阶段 | 内容 |
|------|------|
| 学习试次 | 10 次（左/右各5），含虚线路径引导 |
| 正式实验 | 6 Block × 25 Trial = 150 试次 |
| 单试次 | CUE(2s) → 准备(1s) → 开始(1s) → 想象(4s) → 反馈(4s) → 定格(1.2s) |
| 静息 | 实验前闭眼静息 90s，Block 间休息 15s |

## 运行环境

- **Python** 3.11+
- **依赖**：`pygame`
- **并口驱动**：[InpOut32](http://www.highrez.co.uk/downloads/inpout32/)（可选，无并口时以纯日志模式运行）

## 运行方式

```bash
cd main
python mario_v3_0_7.py
```

### 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开始实验 / 跳过休息 |
| `P` | 跳过当前 Block |
| `R` | 延长休息时间 |
| `E` | 结束实验 |
| `ESC` | 退出程序 |
